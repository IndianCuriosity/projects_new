# Agentic AI Mastery — Full Code Companion

This expanded pack maps each roadmap phase to runnable code examples.

## Install

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -U langchain langchain-openai langchain-core langchain-community langchain-chroma langgraph langsmith chromadb pydantic python-dotenv pandas numpy fastapi uvicorn
```

Create `.env`:

```bash
OPENAI_API_KEY=your_key
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY=your_langsmith_key
LANGCHAIN_PROJECT=agentic-ai-mastery
```
